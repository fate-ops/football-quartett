# Quartett Thema: Fussball(WM)

## Karten Aufbau:
- Photo
- Name
- nation
- Goals
- Assists
- Minutes Played
- Zweikämpfe
- Trefferquote
- rating

## User Stories

### RundenStart
Als Spieler will ich Meine oberste Karte sehen und wenn ich drann bin einen Stat auswählen können um mit meinem Gegner zu vergleichen

-> oberste Karte wird angezeigt
-> ich kann einen der Werte auswählen
-> es wird der gleiche Wert beim Gegner angeschaut und verglichen

### Unentschieden
Wenn der Ausgewählte Stat genau gleich ist, muss ich wenn ich am Zug bin, einen neuen Stat auswählen der verglichen wird

-> Stats sind exakt gleich und somit gibt es keinen Gewinner
-> Ich kann wieder einen der Werte auswählen ausser der von vorher
-> es wird wieder der gleiche Wert beim Gegner angeschaut und verglichen

### RundenEnde
Wenn ich gewinne will ich die Karte vom Gegner nehmen und meine aktuelle Karte und die vom Gegner unten rein tun

-> Der Wert der grösser ist gewinnt
-> beide Karten werden hinten beim Gewinner hinten im Stapel angereiht (zuerst Verlierer- dann Gewinner -Karte)
